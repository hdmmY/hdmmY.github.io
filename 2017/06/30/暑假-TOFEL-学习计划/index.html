暑假 TOFEL 学习计划

发表于 2017-06-30 | 分类于 2017-06 |

这个暑假打算好好学习学习英语，将 TOFEL 考试给考过，以下是我暑假 7-8 月的一个月时间的英语学习计划。

考试和题型

考试由四部分组成：阅读（Reading）、听力（Listening）、口语（Speaking)、写作（Writing）

  • 每部分满分30分，整个试题满分120分，考试时间3-4个小时，阅读约为1个小时，听力时间50分钟，口试时间20分钟，写作时间1个小时。
  • 阅读和听力可能遇到加试，口语和写作没有加试。
  • 考试时应提前半个小时到考场，一般是8点半进场，9点开考。
  • 考试时携带有效证件和从官网打印的报名确认回执。



阅读方面（Reading）
阅读共有 3 篇文章，700-800 单词/篇，12-14 个题目/篇

  • 文章分类
    • 自然科学（地质、人文）
    • 生物科学（植物、动物、生态、环境）
    • 社会科学（艺术、建筑、生理/心理、历史/考古……）
  • 备考
    • 每天至少做两篇阅读
    • 按题型练习，每天至少做完一篇 TPO
    • 自己控制时间，20分钟完成一整篇的练习
  • 注意事项
    • 除了最后一道试题外，其他的试题都是针对文章某一部分进行提问，试题出现的顺序和段落顺序一致。
    • 在做题时养成分段阅读的习惯



听力方面（Listening）
听力共有 2-3 段对话，4-6 个讲座(lecture)，考试时间为 40-60 分钟。

特点：

  • 主要测试考生能否听懂北美和其他英语国家人的英语口语
  • 考试尽量采用真实的学术语言
  • 每段对话后有5个问题，讲座后会有6个问题
  • 整个听力考试过程中，考生都可以记笔记
  • 在准备和答题时，屏幕上会显示倒计时的时钟(可关闭）



听写练习：

  • 听写1分钟以内的材料（重点听语音和语速），科学60s是不错的材料
  • 反复听背词汇书的音频（特别是相应的分类词汇)，跟读和模仿
  • 关键词听写训练，在听完录音后写出录音中出现的关键词



写作方面
综合写作 + 独立写作 (Integratedwriting + Independentwriting)

  • 综合写作：就某一话题阐述自己的观点，300 字以上

  • 独立写作：首先阅读一篇文章，五分钟以后，文章隐去，播放一段与文章有关的课堂演讲。课堂演讲列举了一些论据反驳文章中的论点、论据。随后要求考生在20分钟内写一篇作文，总结课堂演讲的论点、论据，并陈述这些论点、论据是如何反驳文章的论点、论据的，字数要求为150字到225字之间。在写作时，文章会重新显示在屏幕上。这篇作文不要求考生阐述自己的观点。

  • 主要考察：用英文论述和阐述观点的能力



练习方法：

  • 每个星期写两篇文章
  • 每天抄写范文

词汇

我目前的单词水平，应该是在四级和六级之间，应该说和考试要求的词汇量还是有不少差距。新托福阅读的最低要求词汇量是8千左右。

单词计划：

  • TOFEL核心词汇21天突破 - 2轮
  • 平时注意记忆在试卷中出现过的高频单词。
